Try the Demo

The full application is a Flask web app with React frontend. To run it locally, clone the repository and follow the setup instructions.

Quick Start

git clone https://github.com/mvijaykumar2820/image-to-text.git
cd image-to-text
pip install -r requirements.txt
python app.py

Then open http://localhost:5003 in your browser
